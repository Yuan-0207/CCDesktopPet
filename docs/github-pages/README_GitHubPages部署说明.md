# GitHub Pages 5分钟部署说明

## 1. 创建仓库

- 在 GitHub 新建仓库，例如：`ccpet-site`
- 可设为 Public（建议，审核访问更稳定）

## 2. 上传网页文件

把当前目录下 3 个文件上传到仓库根目录：

- `index.html`
- `privacy.html`
- `terms.html`

## 3. 开启 Pages

- 进入仓库 `Settings` -> `Pages`
- `Build and deployment` 选择：
  - Source: `Deploy from a branch`
  - Branch: `main` / `(root)`
- 保存后等待 1-3 分钟

## 4. 获取官网链接

会生成网址，形如：

`https://<你的GitHub用户名>.github.io/ccpet-site/`

## 5. 提审时填写

- 应用官网：填上面的链接
- 隐私政策：`https://<你的GitHub用户名>.github.io/ccpet-site/privacy.html`
- 用户协议：`https://<你的GitHub用户名>.github.io/ccpet-site/terms.html`

## 6. 提交前最后检查

- 手机 4G 下打开官网和两个子页都正常
- 首页“下载方式”替换成你可访问的 APK 下载链接
- 首页截图区域替换为真实截图（至少 4 张）
