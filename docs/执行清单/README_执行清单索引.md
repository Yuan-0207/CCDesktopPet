# 执行清单索引（从这里开始）

本目录存放“正式上市路线”的**详细执行清单**，按周拆分，方便直接照着做与验收。

## 总览

- `00_项目现状与发布门槛.md`：当前差距与发布门槛（P0 阻塞项、上线必备项）
- `01_第1周_M0_审核堵点清零.md`：官网/流程图/合规入口/登录可诊断化
- `01A_官网提交文案_可直接使用.md`：官网页面可直接粘贴文案
- `01B_微信审核流程图_文本版.md`：审核流程图节点与 Mermaid 版本
- `02_第2周_M1-1_登录链路稳定化.md`
- `03_第3周_M1-2_核心体验质量周.md`
- `04_第4周_M1-3_商城产品化.md`
- `05_第5周_M2-1_支付后端底座.md`
- `06_第6周_M2-2_支付前端接入.md`
- `07_第7周_M3_留存系统最小版.md`
- `08_第8周_M4_上市冲刺.md`

## 使用方式（建议）

- 每周开始：先读本周清单，确认“交付物清单”和“验收标准”
- 每天：跑一次真机回归（每周清单里有 10 分钟版）
- 每周结束：对照验收标准打勾，未达标项顺延并记录原因
